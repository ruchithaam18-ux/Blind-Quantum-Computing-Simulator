Blind Quantum Computing Simulator 

Overview
This project explores the concepts of Blind Quantum Computing (BQC) and quantum cybersecurity through a simulation environment.
Blind Quantum Computing enables a client to delegate quantum computations to a quantum server while keeping the computation private.
The project demonstrates concepts such as quantum state preparation, secure quantum communication, eavesdropping detection, and verification of quantum computation.

Features
- Quantum circuit simulation using Qiskit
- Blind quantum computation workflow
- Quantum one-time pad concept
- Eavesdropping attack simulation
- Quantum integrity verification
- Security visualization dashboard
  
Technologies Used
- Python
- Qiskit
- Qiskit Aer
- Streamlit
- Matplotlib

Project Architecture
Client (Alice)
Encrypted Quantum Information
Quantum Server (Bob)
Verification
Eavesdropper (Eve) attempts interception
